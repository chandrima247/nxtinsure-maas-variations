White-Label System
One codebase.
Any partner brand.
Each variation uses CSS custom properties keyed to a data-partner attribute on the root element. Switching the brand takes a single line of JavaScript — colors, CTAs, payment labels, and partner logos all update instantly. Partner preference persists across sessions via localStorage.
